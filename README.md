# DPIA Legal Technology Project

A lightweight and professional front-end shell for a legal-technology group project in **JUS5671 – Legal Technology: Artificial Intelligence and Law**.

The project presents a serious public-facing website for a future **DPIA (Data Protection Impact Assessment)** tool. The current version is intentionally simple: it is built with plain HTML, CSS and JavaScript, works without build tools, and is easy to run locally or publish through GitHub Pages.
